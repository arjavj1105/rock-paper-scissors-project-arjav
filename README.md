# Rock Paper Scissors - Simple Edition

A minimalist, high-performance Rock Paper Scissors game built with React and Vite. Focused on clean UI/UX and senior-level architectural patterns.

## Features
- **Clean UI**: Simple white theme with focused interactions.
- **Score Tracking**: Persistent session scoring for player and AI.
- **History Log**: Detailed battle logs for the current session.
- **Pure Game Logic**: Isolated game mechanics for stability and predictability.

## Tech Stack
- **Frontend**: React 18+
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (CSS3)
- **Linting**: ESLint with React Purity Rules

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Development
This project follows strict React purity guidelines. Game logic is separated from the render cycle to ensure high performance and zero side-effect rendering.
